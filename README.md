# Employee Data Analysis using Pandas

## Project Description

This project focuses on analyzing employee data using Python and Pandas. The dataset contains employee details such as salary, team, gender, joining date, bonus percentage, and management status.

The project demonstrates data cleaning, data manipulation, exploratory data analysis, grouping, aggregation, and business insight generation using Pandas.

## Objectives

* To perform data cleaning and preprocessing
* To analyze employee salary and bonus information
* To understand team-wise and gender-wise employee statistics
* To generate business insights using data analytics techniques

## Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook / Google Colab
* Git & GitHub

## Features

* DataFrame and Series creation
* Data inspection and statistical analysis
* Missing value handling
* Duplicate data removal
* String operations
* Datetime operations
* GroupBy and aggregation analysis
* Team-wise salary analysis
* Gender-based salary comparison
* Business insight generation

The dataset contains the following attributes:

* First Name
* Gender

* Last Login Time
* Salary
* Bonus %

## Analysis Performed

* Employee data exploration
* Salary distribution analysis
* Team-wise average salary calculation
* Bonus amount computation
* Missing value treatment
* Datetime feature extraction
* Team and gender-based grouping
* Highest earning team identification
